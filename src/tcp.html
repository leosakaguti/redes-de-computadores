Modelo TCP/IP

O modelo TCP/IP, em contraste com o modelo OSI, é uma abordagem mais enxuta e pragmática para a arquitetura de redes de computadores. Enquanto o modelo OSI possui sete camadas distintas, o modelo TCP/IP é composto por quatro camadas principais: a camada de Link, a camada de Internet, a camada de Transporte e a camada de Aplicação. Essa simplicidade é uma das razões pelas quais o modelo TCP/IP se tornou amplamente adotado e a espinha dorsal da internet.

Camadas

Aplicação

É onde os aplicativos se comunicam com a rede.

Transporte

Garante a entrega confiável de dados. Exemplo: Transferência de arquivos (usando TCP) e chamadas de voz via Internet (usando UDP).

Rede

Roteia pacotes de dados entre redes. Exemplo: Roteadores na Internet usam o protocolo IP para encaminhar pacotes.

Estrutura Física

Lida com a comunicação dentro de uma rede local. Exemplo: Redes Ethernet ou conexões Wi-Fi, onde dispositivos se conectam diretamente.

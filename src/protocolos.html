Protocolos

Qualquer usuário pode enviar e receber mensagens instantâneas, compartilhar arquivos em seu site e acessar diversos domínios na web sem precisar aprender línguas estrangeiras. Os protocolos de internet agem como uma "língua universal" entre computadores, permitindo a comunicação sem necessidade de software adicional, independentemente do fabricante ou sistema operacional. Isso possibilita a comunicação sem problemas entre computadores na rede mundial de computadores.

Tipos

Camada de Aplicação

WWW (navegação web), HTTP, SMPT (emails), FTP (transferência de arquivos) e SSH. Usada pelos programas para enviar e receber dados de outros programas pela própria internet

Camada de Transporte

TCP, UDP e SCTP. Para transporte de arquivos recebidos da camada anterior. Aqui acontece a organização e a transformação deles em pacotes menores, que serão enviados à rede.

Camada de Rede

IP (IPv4 e IPv6). Os arquivos empacotados na camada anterior são recebidos e anexados ao IP da máquina que envia e que recebe os dados. Daqui, são enviados pela internet usando a próxima camada.

Camada de Estrutura Física

Ethernet e Modem. É a camada que executa o recebimento ou envio de arquivos na web.

Protocolos

TCP/IP

Dentre todos os protocolos de rede, juntos, eles formam a base de envio e recebimento de dados por toda a internet.

HTTP

Ele é o mais básico e usado para navegação em sites da internet.

HTTPS

O protocolo HTTPS é e funciona de forma exatamente igual ao HTTP. A diferença da letra “S” na sigla é uma camada extra de proteção, indicando que sites e domínios que possuem esse protocolo são seguros para o usuário acessar.

DHCP

Ele permite que os computadores consigam um endereço de IP automaticamente.

FTP

Ele surgiu antes mesmo do padrão TCP/IP, que é a base das conexões de internet. E é o modo mais simples de transferir dados entre duas máquinas pela rede.

SFTP

Ele é, basicamente, o protocolo FTP com uma camada de proteção a mais aos arquivos transferidos.

SSH

É um dos protocolos específicos de segurança de troca de arquivos entre cliente e servidor.

POP3

Ele é usado para mensagens eletrônicas, ou seja, os populares emails.

SMTP

Voltado para o envio de mensagens eletrônicas (emails).

IMAP

Assim como os dois anteriores, o protocolo IMAP também é voltado para envio e recebimento de emails.

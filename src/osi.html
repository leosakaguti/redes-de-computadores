Modelo OSI

Antes do modelo OSI, as redes de computadores usavam protocolos proprietários e diferentes sistemas de comunicação, o que dificultava a interoperabilidade entre diferentes dispositivos e sistemas. A necessidade de uma padronização levou ao desenvolvimento do modelo OSI, que foi desenvolvido pela Organização Internacional para Normalização (ISO) na década de 1970 e foi adotado como um padrão internacional para protocolos de comunicação, sendo composto por sete camadas distintas, cada uma com funções específicas que contribuem para a comunicação eficiente entre sistemas. Cada camada se comunica com a camada correspondente em outro sistema, seguindo um conjunto de protocolos e padrões definidos.

Camadas

Física

Esta camada lida com a transmissão física dos dados, como os meios de comunicação, a voltagem elétrica e as características mecânicas.

Enlace

A camada de enlace é responsável pela transmissão de dados confiável entre dispositivos adjacentes e pela detecção e correção de erros.

Rede

A camada de rede gerencia a comunicação de dados entre redes diferentes, incluindo o roteamento, a fragmentação e a montagem de pacotes.

Transporte

A camada de transporte garante a entrega confiável de dados entre os sistemas finais, dividindo e reagrupando as informações em segmentos.

Sessão

A camada de sessão estabelece, gerencia e encerra as conexões entre os sistemas finais, permitindo a comunicação contínua entre eles.

Apresentação

A camada de apresentação lida com a formatação e a representação dos dados, garantindo que os sistemas finais possam interpretar e exibir as informações corretamente.

Aplicação

A camada de aplicação é responsável por fornecer serviços de rede para os aplicativos do usuário, como e-mail, navegação na web e transferência de arquivos.
